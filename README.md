# 🍅 番茄的旅程

一款以番茄为主角的重力感应网页游戏，通过三章体验：**生长 → 运输 → 冰箱**，让玩家体会食物从田间到餐桌的来之不易。

## 玩法说明

- **第一章 · 生长**：刚发芽的番茄需要水分和阳光。倾斜手机左右摇摆，让植株碰到左侧的💧水滴和右侧的☀️阳光，在限时内把生长条养满，否则会被当作次品丢弃。
- **第二章 · 运输**：番茄在颠簸的车厢里。用重力感应保持番茄在绿色安全区内，避免磕碰导致损伤值爆表，否则会被扔掉。
- **第三章 · 冰箱**：番茄被买回家放进冰箱，冰箱像一座迷宫。倾斜手机让番茄滚动到绿色出口，才能被做成菜端上桌；否则会遗忘在角落腐烂。

失败时会看到垃圾桶里那些没能被珍惜的番茄。

## 如何运行

1. 用**手机**打开效果最佳（需要重力感应）。
2. 在电脑上也可用浏览器打开，支持**键盘**模拟倾斜：
   - **A / D** 或 **← / →**：左右
   - **W / S** 或 **↑ / ↓**：上下（第二章、第三章）

### 本地运行

- 直接用浏览器打开 `index.html` 即可。
- 若部署到服务器，请使用 **HTTPS**，否则在手机上可能无法使用重力感应（部分浏览器会限制）。

### 手机权限

- iOS：点击「开始旅程」后，若弹出「运动与方向」权限，请选择**允许**。
- Android：如遇权限提示，请允许后再玩。

## 技术说明

- 纯前端：HTML5 + CSS + JavaScript，无需后端。
- 重力感应：`DeviceOrientation` / `DeviceMotion` API。
- 绘制：Canvas 2D。
